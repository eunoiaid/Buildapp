Pengantar HTML dan CSS

Diposting oleh Admin pada 24 Juli 2025

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web dan aplikasi web. Dengan HTML, kita bisa menyusun struktur konten web.

CSS (Cascading Style Sheets) adalah bahasa style sheet yang digunakan untuk menjelaskan presentasi dokumen yang ditulis dalam bahasa markup seperti HTML. CSS mendeskripsikan bagaimana elemen-elemen harus ditampilkan di layar, di atas kertas, atau di media lain.

Dengan menggabungkan keduanya, kita dapat menciptakan halaman web yang terstruktur dengan baik dan terlihat menarik. HTML menyediakan kerangka, sementara CSS memberikan 'gaya' pada kerangka tersebut.
